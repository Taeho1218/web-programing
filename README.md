# 손말잇기 (정적 HTML)

이 폴더는 `index.html`, `styles.css`, `data.js`, `app.js`만으로 동작합니다.

## 실행 방법

- **파일로 열기**: `index.html`을 브라우저에서 연다. (일부 기능은 로컬 파일에서는 카메라/API 제한이 있을 수 있음)
- **간단 서버 권장** (프로젝트 폴더에서):

```bash
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속.

외부 CDN: jQuery, jQuery UI 스타일/스크립트가 필요하므로 **인터넷 연결**이 있어야 합니다.
